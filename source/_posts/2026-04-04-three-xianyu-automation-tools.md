想用闲鱼赚钱搞副业？GitHub 上真正靠谱的闲鱼开源工具就这几个。别人还在手动刷页面、错过好货或累死累活回消息，你直接用这些工具就能实现：

- ✅ 24小时智能秒杀捡漏好货
- ✅ AI 自动筛选性价比商品
- ✅ AI 客服智能自动出单
- ✅ 多账号批量管理

---

## 1️⃣ 闲鱼智能监控工具：ai-goofish-monitor

**功能：**
能实时/定时监控闲鱼商品，自动筛选产品，通过 AI 分析性价比和商品真实度，帮你快速发现低价好货，捡漏秒拍必备。

🔗 GitHub 地址：[github.com/Usagi-org/ai-goofish-monitor](https://github.com/Usagi-org/ai-goofish-monitor)

---

## 2️⃣ 闲鱼 AI 智能客服机器人：XianyuAutoAgent

**功能：**
- 7×24 小时自动化回复/出单
- 支持多专家协同决策
- 智能议价和上下文感知对话
- 卖家自动处理买家询价、砍价和出单，大幅减少手动聊天时间

这个项目已经有 **6.2k Stars**，是目前 GitHub 上星标最多的闲鱼自动化项目。

🔗 GitHub 地址：[github.com/shaxiu/XianyuAutoAgent](https://github.com/shaxiu/XianyuAutoAgent)

---

## 3️⃣ 闲鱼多账号智能管理系统：xianyu-auto-reply-fix

**功能：**
- 支持多账号管理
- AI 自动回复消息
- 自动发货确认
- 多渠道消息通知
- 提供完整 Web 管理后台
- 适合系统化/矩阵化运营闲鱼店铺，大幅提升出单效率

🔗 GitHub 地址：[github.com/GuDong2003/xianyu-auto-reply-fix](https://github.com/GuDong2003/xianyu-auto-reply-fix)

---

## 总结

这三个工具社区活跃、功能成熟，**大多支持 Docker 一键部署**，新手也容易上手：

| 工具 | 用途 |
|------|------|
| ai-goofish-monitor | 买家捡漏，监控秒杀好货 |
| XianyuAutoAgent | 卖家AI自动回复客服 |
| xianyu-auto-reply-fix | 多账号管理，矩阵运营 |
