# Windows 11 激活教程

> ⚠️ **重要声明**：本教程仅供学习参考，请支持正版软件。请在购买正版授权后使用。

## 方法一：使用命令行激活（适合有密钥的用户）

### 1. 以管理员身份打开命令提示符
- 按下 `Win + X` 键
- 选择 "Windows 终端(管理员)" 或 "命令提示符(管理员)"
- 在弹出的用户账户控制中选择 "是"

### 2. 输入命令安装密钥
```cmd
slmgr.vbs /ipk 你的产品密钥
```

### 3. 设置激活服务器
```cmd
slmgr.vbs /skms kms.v0v.bid
```

### 4. 执行激活
```cmd
slmgr.vbs /ato
```

### 5. 查看激活状态
```cmd
slmgr.vbs /dlv
```

## 方法二：使用KMS一键激活脚本

### 1. 下载激活脚本
- 打开浏览器访问：https://github.com/massgravel/Microsoft-Activation-Scripts
- 点击绿色的 "Code" 按钮 → 选择 "Download ZIP"

### 2. 解压文件
- 下载完成后解压ZIP文件到任意目录

### 3. 执行激活
- 打开解压后的文件夹
- 右键点击 `All-In-One-Version` 目录下的 `KMS.cmd`
- 选择 "以管理员身份运行"
- 按照提示选择对应的选项（通常选择 1 即可自动激活）

### 或者使用在线命令（更简单）：
1. 以管理员身份打开PowerShell
2. 复制粘贴以下命令：
```powershell
irm https://massgrave.dev/get | iex
```
3. 按回车执行，然后按照界面提示选择激活Windows即可

## 方法三：数字权利激活

对于Windows 10免费升级过来的Windows 11，通常会自动绑定硬件数字权利，重新安装后联网自动激活：
1. 安装Windows 11时跳过输入密钥步骤
2. 安装完成后连接网络
3. 系统会自动根据硬件激活

## 检查激活状态

1. 按下 `Win + I` 打开设置
2. 依次点击 **系统** → **激活**
3. 查看激活状态显示是否已激活

## 常见问题

### Q: 激活后可以更新吗？
A: KMS激活有效期一般为180天，到期后可以重新激活，不影响系统更新。

### Q: 这个安全吗？会不会有病毒？
A: Massgrave的脚本是目前最常用的，社区验证过没有恶意代码，但仍然建议你使用正版。

### Q: 提示激活失败怎么办？
A: 检查网络连接，尝试更换KMS服务器，或者使用Massgrave脚本自动处理。
