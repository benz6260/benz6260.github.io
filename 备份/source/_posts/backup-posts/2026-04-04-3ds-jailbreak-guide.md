## 当前推荐方案：Boot9strap (B9S) + Luma3DS

B9S 是目前最完美的破解方式，替代了旧版 A9LH，支持全机型，启动快，功能完整，永久固化不会掉。

### 破解前提

- 如果你的系统版本 **≤ 11.17**：可以用浏览器漏洞软破，不需要额外硬件，最简单
- 如果你的系统版本 ≥ 11.18：需要用 `ntrboot` 硬破，需要一张兼容的 DS 烧录卡

## 方法一：浏览器软破（推荐，系统 ≤ 11.17）

### 准备工作
- 一张可用的 SD 卡（根据你的机型：
  - 老 3DS：最大支持 32GB
  - 新 3DS：支持 64GB / 128GB，需要格式化为 FAT32
- 下载破解工具包：可以从 [3DS 官方破解指南](https://3ds.hacks.guide/) 下载最新版

### 破解步骤

1. **准备 SD 卡**
   - 格式化 SD 卡为 FAT32，分配单元大小 32KB
   - 把破解工具包所有文件解压到 SD 卡根目录

2. **修改DNS**
   - 进入「系统设置」→「Internet」→「Network Settings」
   - 选择你的 WiFi → 「Change Settings」→ 「DNS」→ 「Manual」
   - 设置 DNS 为 `94.141.117.129`，保存

3. **触发浏览器漏洞**
   - 打开 3DS 内置浏览器
   - 访问地址：`http://pig.fish`
   - 按照页面提示操作，会自动进入破解引导程序

4. **安装 Boot9strap + Luma3DS**
   - 进入引导后，按 A 键安装 B9S
   - 安装完成后，按 Start 重启机器
   - 开机自动进入 Luma3DS 配置，直接按 Start 启动系统

5. **安装 FBI**
   - 重启后进入系统，打开 Luma3DS 配置
   - 使用 SafeMode 安装 FBI 到 home menu
   - 现在你就可以安装 CIA 格式游戏了

## 方法二：ntrboot 硬破（适用于 11.18+ 系统）

如果你的 3DS 已经升级到最新官方系统无法软破，就用这个方法：

### 准备
- 一张兼容 ntrboot 的 DS 烧录卡（比如 R4i Gold 3DS Plus）
- 提前在电脑上把 ntrboot 写入烧录卡
- 剩下步骤和软破后面一样，安装 B9S + Luma3DS

### 优点：
- 不依赖系统版本，任何版本都能破
- 成功率 100%
- 只需要烧录卡，不需要拆机

## 破解完成后的配置

### 推荐安装这些工具：
| 工具 | 作用 |
|------|------|
| **FBI** | 安装/删除 CIA 格式游戏 |
| **GodMode9** | 文件管理，备份NAND，dump游戏卡 |
| **Anemone** | 安装主题 |
| **Checkpoint** | 游戏存档备份/还原 |
| **Luma Updater** | 在线更新 Luma3DS |

### 备份很重要！
破解第一步请用 GodMode9 备份你的 **原厂 NAND**，备份文件放到电脑保存，以后出问题可以恢复。

## 安装游戏

游戏格式分两种：
1. **CIA 格式**：可以安装到主菜单，和正版一样显示图标，推荐
   - 打开 FBI → 「SD」→ 选择你的 .cia 文件 → 「Install and delete CIA」
2. **.3ds 格式**：需要用网关或者破解卡带模式运行，现在少用

安装完按 Home 键回到主菜单就能看到游戏了。

---

## 注意事项

1. **不要更新官方系统！** 更新会覆盖破解，需要重新破解。可以用 Luma3DS 的更新功能更新游戏卡带补丁。
2. **联网风险**：如果要进 eShop 或者联网玩，建议设置 Luma 禁用 SSL 检测，有极小概率被 BAN。
3. **转移破解**：换 SD 卡只需要复制所有文件过去就行。

## 常见问题

### Q: B9S 和 A9LH 有什么区别？
A: B9S 是新一代破解，更稳定、更快、安装简单，直接用 B9S 就行，不需要再转 A9LH。

### Q: 破解后正版卡带还能用吗？
A: 完全可以，破解系统兼容正版卡带，可以正常玩。

### Q: 什么是虚拟系统？我需要做吗？
A: B9S 时代不需要虚拟系统了，直接真实系统破解就很好用。
